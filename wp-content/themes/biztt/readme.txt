== Description ==
BizTT is an multipurpose business WordPress themes which is made for stock exchange, stock market, business, finanace and many more.

== Frequently Asked Questions ==

= How to customize theme ? =

Just click on Appearance -> Customize
Now you can customize all section in theme

= How to Upgrade theme to pro ?=

Just click on Appearance -> Customize 
Now find upgrade to pro button, just click on it to upgrade pro version

= How to work on slider? =

Just click on Appearance -> Customize -> BizTT Home Slider

= How to work on Resource Box? =

Just click on Appearance -> Customize -> BizTT HomePage Business Box

= How to work on Footer Widgets ? =

Just click on Appearance -> Customize -> Widget

= How to work on Footer copyright ? =

Just click on Appearance -> Customize -> BizTT Footer
